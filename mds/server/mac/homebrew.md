# Mac Homebrew 筆記

## 安裝

* `ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

	![](img/homebrew-01.png)

* 安裝 git，`brew install git`

	![](img/homebrew-02.png)

* 更新 Homebrew，`brew update`

	![](img/homebrew-03.png)

* `brew tap homebrew/dupes`

	![](img/homebrew-04.png)

* 安裝 apple-gcc，`brew install apple-gcc42`

	![](img/homebrew-05.png)
