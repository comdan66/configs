# iTerm2 常用設定

## preferences

* 路徑 `preferences...`，或按 `command + ,`

### Keys
| keyboard | Action        | value  |
| -------- | ------------- | ------ |
| ⌘→       | Send Hex Code | 0x05   |
| ⌘←       | Send Hex Code | 0x01   |
